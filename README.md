# Gravitate
> Project scaffolding for Grav.

## Features

- Pre-configured Grav
- SCSS compilation
- Babel ES6 support

## Installation

    npm install gravitate -g

## Usage

Generate a new Grav site.

    $ gravitate new

## Building locally

For testing purposes you can `cd` into the directory and install it from there.

    npm install . -g
